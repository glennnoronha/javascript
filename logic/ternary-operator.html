Ternary Operators

Take a look at this block of code:

let age = 21;
let message;

if (age >= 18) { message = "You are an adult";}
else { message = "You are not an adult"}

console.log(message);

As programmers, we want to avoid rewriting code. This is why ternary operators exist. We can write the above code much faster using ternary operators:

let message2 = age >= 18 ? "You're an adult" : "You're not an adult";
console.log(message2);

Here's a ternary operator with more than 2 conditions

age = 15;
message = age >= 65 ? "You're a senior" :
        age >= 18 ? "You're an adult" :
        "You're a minor";

console.log(message);
