If Statements

This is what an if statement looks like in javascript

let age = 22;

if (age > 18) {
    console.log("You are an adult")
}

Since the condition is true, it will be outputted to the console

Now let's add an else to our if statement

age = 15;

if (age > 18) {
    console.log("You are an adult")
} else {
    console.log("You are not an adult")
}

Since the age is now 15 and the if statement is false, it will now go to the else block

You can also do else if statements in JavaScript

let temperature = 30;

if (temperature > 30) {
    console.log("It's hot outside.");
} else if (temperature > 20) {
    console.log("The weather is warm.");
} else {
    console.log("It's cold outside.");
}

Since temperature is greater than 20, it will output the else if block

You can also do nested if statements in JavaScript

age = 20;
let hasTicket = true;
