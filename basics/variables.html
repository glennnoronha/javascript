Variables

Declaration and Assignment

Declaring a variable

let x;

Assigning a value to a variable

x = 10;

You can also do declaration and assignment together

let x = 10;
console.log(x);

Variable Types

Here are the different types of variables in JavaScript

let age = 25;
let name = 'Glenn';
let graduated = true;
      
console.log(age, typeof age);
console.log(name, typeof name);
console.log(graduated, typeof graduated);

Putting Variables into a Webpage

Let's put some of the variables into the webpage

HTML code:

<p id="myFullName"></p>
<p id="country"></p>
<p id="birthYear"></p>
<p id="student"></p>

JavaScript code:

let fullName = "Glenn Noronha";
let country = "Canada";
let birthYear = 2002;
let student = false;

document.getElementById('myFullName').textContent = `My name is: ${fullName}`;
document.getElementById('country').textContent = `I am from: ${country}`;
document.getElementById('birthYear').textContent = `I was born in: ${birthYear}`;
document.getElementById('student').textContent = `I am a student: ${student}`;
