User Input

Easy Way: window prompt

let username = window.prompt('What is your username?');
console.log(username);

Profesessional Way: HTML textbox

HTML code:

<h2 id="myH2">Profesessional Way: HTML textbox</h2>
<label>Username:</label>
<input id="myText" type="text">
<button id="mySubmit">Submit</button><br>

JavaScript code:

let username;

document.getElementById('mySubmit').onclick = function() {
    username = document.getElementById('myText').value;
    console.log(username);
    document.getElementById('myH2').textContent = `Hello, ${username}`
}

When we click submit, the username is logged to the console, and we modify the text of the h2 element
