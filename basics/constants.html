Constants

Declaring a Constant

To declare a constant you can use const:

const PI = 3.14159;
console.log(PI);

Let's try changing the value of PI:

PI = 3.14;
console.log(PI);

Constant Objects

  
You can use const with objects too:

  
const person = {
  name: "Glenn",
  age: 22
};
console.log(person);

  

  
You can change properties of a const object:

  
person.age = 23;
console.log(person);
